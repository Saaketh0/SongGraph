# Graph Expansion Rules (Planned V1)

## Core Rules
- Expand from selected visible node(s)
- Add only unseen nodes
- Add only unseen edges
- Stop when visible node cap (1000) is reached

## UX Rules
- Disable expansion controls when limit reached
- Show clear limit-reached status
- Support reset/recenter workflows

## Phase 0 Status
Rules documented only; implementation deferred to graph service + frontend graph state phases.
